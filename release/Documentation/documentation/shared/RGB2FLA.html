Syntax:

    I=RGB2FLA(IMAGE);

Description:

    Converts an RGB image into an FLA (flat) data structure.

Input:

    IMAGE: RGB of gray-scale image contained in a cols x rows x 3 array.

Output:

    I: Structure containing the FLA data structure. This is designed to be
    consistent with that delivered by FLAread.

Example:

    IMAGE = imread('test.jpg');
    I = rgb2fla(IMAGE);
    HSZ = Scyllarus(I);

See also:

    Scyllarus, FLAwrite, FLAread
